# Goal — Obsidian Learning System

> "Do you like the way you learn right now?
> How do you want to learn in your life?"

## The Problem
There is too much to learn and too many resources on the internet.
This makes it hard to:
- Know what is actually worth learning
- Stay focused instead of jumping between topics
- Keep track of what I have already learned
- Reduce the friction and decision fatigue during the learning process

Resources come in all formats — videos, articles, podcasts, courses —
and manually filtering through them takes too much time and energy.

## What This System Should Do
- Use AI to filter and summarize resources (articles, videos) so I can
  quickly decide if something is worth going through
- Reduce friction at every step of learning — finding, processing,
  and storing knowledge
- Manage and organize what I have learned so I never lose track of it
- Automate as much of the learning process as possible

## What Success Looks Like
I spend less time deciding what to learn and more time actually learning.
My notes are organized, connected, and easy to retrieve.
Learning feels like a system I trust, not a pile I am drowning in.
