# 【生成式AI時代下的機器學習(2025)】第七講：DeepSeek-R1 這類大型語言模型是如何進行「深度思考」（Reasoning）的？

## Description

投影片：https://docs.google.com/presentation/d/1lQwn837sXA9_sN0YFuPWScwFETbNbe6e/edit?usp=sharing&ouid=115046073158939078465&rtpof=true&sd=true

這堂課介紹了四種讓語言模型能夠進行深度思考的方法：
(1) 更強的思維鏈（Chain-of-Thought，CoT）
(2) 為模型提供推理工作流程
(3) 教導模型推理過程（模仿學習，Imitation Learning）
(4) 以結果為導向學習推理（強化學習，Reinforcement Learning，RL）（這是 DeepSeek-R1 主打的方法，雖然其他三種在 DeepSeek-R1 的訓練過程中也有使用）
