
This file provides guidance to Claude Code (claude.ai/code) when working with documents in this repository.
## What this repo is

This is a personal Life Infrastructure workspace — an Obsidian vault and Claude Code project in one. The goal is to build a framework and infrastructure for my life, create an AI assistant to perform the tasks automatically. Currently I only have 2 systems that I am working on. 

- **Learning System** — Manage critical knowledge I have learned, and keep track of future knowledge to learn.
- **Project Management System** — Manage and keep track of current and future projects I am working on. 
## Folder Structure

As stated above, there are currently only two active systems, which are the following 2 directories @Learn and @Projects. Other directories like @Templates and @Scratchpad are experimental folders that are utils or legacy used for development. 

## Working Style

- DO NOT REMOVE OR DELETE ANY FILE
- AVOID REPLACING AND DELETING CONTENT WHEN EDITING FILES, APPEND OR ADD CONTENT INSTEAD
- WHEN MAKING A LOT OF CHANGES TO A FILE SUCH THAT IT HAS A NEW LOOK AND NEW FEELING, CREATE A NEW FILE TO DEMONSTRATE THE DIFFERENCE BETWEEN OLD FILE
## Obsidian plugins in use

- **Templater** — template engine for note creation scripts
- **Folder Notes** — allows folders to have an associated note
- **obsidian-local-rest-api** — exposes vault over local REST API
- **mcp-tools** — MCP server bridge for Claude integration
